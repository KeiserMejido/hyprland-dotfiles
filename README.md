# hyprland-dotfiles
My personal dotfiles for arch. A clean, functional, and aesthetic Hyprland setup.

A workspace focused on clean, visual consistency, and workflow efficiency. Currently running a Gruvbox Material theme over Hyprland.

> **Note:** This repository is constantly evolving as I refine my setup.

## The Setup
| Component | Choice |
|---|---|
| **OS** | Arch Linux |
| **Window Manager** | Hyprland |
| **Status Bar** | Waybar |
| **Color Scheme** | Gruvbox Material |
| **Audio Visualizer** | Cava |
| **App Launcher** | Rofi |
| **Terminal** | Kitty |
| **Shell** | Fish |
| **Prompt** | Starship |
| **Lockscreen** | Hyprlock 

## Features
* **Dynamic Waybar:** Customized modules, a floating calendar, network status, battery status, keyboard language and responsive Cava integration.
* **Smart Window Rules:** Optimized to productivity with scrooling window tabs
* **Modern Terminal Experience:** Kitty terminal paired with Fish shell for intelligent autosuggestions and Starship for a fast, aesthetic, and informative prompt.
